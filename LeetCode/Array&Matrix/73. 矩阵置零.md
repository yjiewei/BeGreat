#### [73. 矩阵置零](https://leetcode-cn.com/problems/set-matrix-zeroes/)

给定一个 m x n 的矩阵，如果一个元素为 0，则将其所在行和列的所有元素都设为 0。**请使用原地算法。**

```java
示例 1:

输入: 
[
  [1,1,1],
  [1,0,1],
  [1,1,1]
]
输出: 
[
  [1,0,1],
  [0,0,0],
  [1,0,1]
]

示例 2:

输入: 
[
  [0,1,2,0],
  [3,4,5,2],
  [1,3,1,5]
]
输出: 
[
  [0,0,0,0],
  [0,4,5,0],
  [0,3,1,0]
]
```


进阶:

一个直接的解决方案是使用  O(mn) 的额外空间，但这并不是一个好的解决方案。
一个简单的改进方案是使用 O(m + n) 的额外空间，但这仍然不是最好的解决方案。
你能想出一个常数空间的解决方案吗



**答题思路：** 直接暴力解法：遍历矩阵记录为0的行列号，最后将它所在的行列全部一次性置为0，

时间O(MN)----空间O(M+N)

另外一种优化空间复杂度的解法是：以第一行和第一列作为标志位，标志那一列或者那一行存在0值，不过这个方法存在一个需要解决的问题，就是最左上角一个位置要同时代表第一行和第一列，只能二选一，另外一个需要用一个变量来存储标志。

下面的写法是将[0][0]位置的元素看做是第一行的标志位，第一列的标志位用col0来标记。

**答题代码：**

```java
class Solution {
    public void setZeroes(int[][] matrix) {
        boolean col0 = false;  // 用来表示第0列是否要置为0
        int i = 0;
        int j = 0;
        for(i = 0; i < matrix.length; i++){
            for(j = 0; j < matrix[0].length; j++){
                // 这里从头开始遍历，先列再行
                // 因为左上角那个位置只能表示一行，如果那一行后面出现0就改那个位置的
                // 但是如果最左上角也是0，需要另外一个变量，来表示说这一列后面要设置为0，否则他代表的是当前行
                if(matrix[i][j] == 0){
                    matrix[i][0] = 0; // 这一行有0值，修改这一行的第一个元素为0
                    if(j == 0) col0 = true; // 第一列的元素，不能作为当前列的修改标准
                    else matrix[0][j] = 0; // 这个是第一行的元素了，代表这一列存在0值，后面这一列要置为0
                }
            }
        }
		// 从下往上遍历，从第二个元素往后遍历，第一个元素动不动得看标志位col0的
        // 如果当前行有0，第一个位置已经在上一个方法置为0了
        for(i = matrix.length-1; i >= 0; i--){
            for(j = 1;j < matrix[0].length; j++){
                // 行列都设为0
                if(matrix[i][0] == 0 || matrix[0][j] == 0){
                    matrix[i][j] = 0;
                }
            }
        }
		// 【0】【0】位置如果是0表示第一列全部都要置为0 第一行已经在上一个方法做过了
        if(col0){
            for(i = 0; i < matrix.length; i++){
                matrix[i][0] = 0;
            }
        }
    }
}
```
