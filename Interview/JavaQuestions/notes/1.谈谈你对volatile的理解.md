### 1.谈谈你对volatile的理解

##### 1.1 它是java虚拟机提供的轻量级的同步机制

- 保证可见性
- **不保证原子性**
-  禁止指令重排序



##### 1.2 JMM模型（java内存模型）（不是真实存在，只是一种规范，比如你属龙）

###### 规范：可见性、原子性、有序性

- 主内存
- 自己的工作内存（变量拷贝）（线程私有）
- 硬盘（主内存） mysql < 内存 redis < 缓存 cache < cpu

当多线程访问并试图修改硬盘里的某个对象变量时，其中一个修改后会写会硬盘并通知其他线程我已经将其修改，这就是变量可见性；
